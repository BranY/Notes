#  机器学习中的偏差和方差

在机器学习领域我们总是希望使自己的模型尽可能准确地描述数据背后的真实规律。一般来说准确，其实就是误差小。在领域中，排除人为失误，人们一般会遇到三种误差来源：**随机误差、偏差和方差**。偏差和方差又与**欠拟合**及**过拟合**紧紧联系在一起。因为随机误差是不可消除的，所以此篇我们讨论在偏差和方差之间的权衡（Bias-Variance Tradeoff）。
